\textbf{猕猴ECoG电极植入的长期稳定性评估} \hfill 期刊在投 \quad 2024.6 -- 2025.12 \titlebreak
项目简述：高密度µECoG能够以亚毫米分辨率探究神经功能，但受术中脑移位影响，电极定位存在数毫米偏差，阻碍电生理信号与功能图谱的精确配准。本研究通过透明颅窗接口实现活体视觉定位，系统评估猕猴ECoG电极植入后的空间位置稳定性与电生理特性稳定性，为长期多模态神经接口研究提供技术基础。
\begin{itemize}[nosep]
    \item 针对脑移位导致的电极定位不确定性，应用基于血管交叉点基准的图像配准算法（透视变换+最小二乘拟合），量化术后14天内电极微位移量（中位0.20 mm）
    \item 建立ECoG信号预处理流程：数字滤波、降采样、共模重参考（CAR）及ICA伪迹剔除，确保信号质量
    \item 采用小波变换进行时频分解，提取gamma频段功率谱特征，频谱白化处理以补偿1/f衰减，增强可视化效果
    \item 构建MVPA解码框架：基于gamma功率时空特征（64通道×8时间点）构造高维特征向量，采用SVM结合10折交叉验证，实现红绿/黑白光栅刺激的高精度分类（>75\%准确率）
    \item 纵向评估5个月以上记录稳定性，验证组织覆盖状态下神经解码性能的持续有效性。
\end{itemize}

\textbf{增强大语言模型生物信息学工具调用能力} \hfill CCF-C类会议 \quad 2025.01 -- 2025.08 \titlebreak
项目简述：针对大语言模型在开放式科学发现中的工具调用局限，本项目构建了生物信息学基准测试Co-BioBench（涵盖15个真实文献场景，3,785个元任务），并提出SDTL微调框架。通过双代理协作数据生成与动态课程学习，实现了7B规模模型在复杂科学工作流上超越GPT-4o的性能。
\begin{itemize}[nosep]
    \item 生物信息学工具命名高度专业化且文档稀疏，现有模型易过拟合于函数名记忆而非语义理解，同时缺乏基于真实文献复现的高质量工具调用训练数据。
    \item 设计多维度数据增强策略：实施函数名随机化、参数名遮蔽及跨域风格迁移，消除模型对表面模式的依赖。
    \item 针对数据构建，检索15篇开放代码的文献，通过Scientist-Co-pilot双代理交互流程提取真实R/Python工具调用场景，完成多轮数据清洗后，利用LLM自动生成负向样本（构造干扰函数及错误参数组合）以提升模型判别能力。
    \item 所构建的数据集支撑SDTL-Qwen2.5-7B模型在Co-BioBench基准上取得51.65\%准确率，较基线提升17个百分点并超越GPT-4o；消融实验表明，数据增强策略显著提升了模型在多样化的科学应用中的鲁棒性，有效解决了科学工具命名歧义带来的泛化难题。
\end{itemize}
